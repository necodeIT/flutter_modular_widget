ModularState

ModularState<T> typedef

ModularState<T> = State<StatefulModularWidget<T>>

Canonical base type for the content State.

Extend this for your content state class (e.g. _MyScreenState) to get the correct widget typing: State<StatefulModularWidget<T>>.

Example:

class _MyScreenState extends ModularState<MyData> {
  @override
  Widget build(BuildContext context) {
    final data = widget.data;
    final repo = widget.get<MyRepo>();
    // ...
    return Container();
  }
}
